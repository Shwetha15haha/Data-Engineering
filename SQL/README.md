# Required Software & Learning Resource

To set up and manage your SQL database, the following software is required:

- **SQL Server Express**: A lightweight server for hosting your SQL database.
- **SQL Server Management Studio (SSMS)**: A GUI for managing and interacting with databases.
- **VS Code Extension**: [MSSQL Extension for Visual Studio Code](https://www.aka.ms/vscode-mssql)

Additionally,

1. the following Udemy course is being followed for learning and guidance:([SQL Bootcamp](https://www.udemy.com/course/the-complete-sql-bootcamp-30-hours-go-from-zero-to-hero/?couponCode=25BBPMXINACTIVE))
2. the following git repos can be referred for projects :
   [sql-data-wrehouse-project](https://github.com/DataWithBaraa/sql-data-warehouse-project),
   [sql-data-analytics-project](https://github.com/DataWithBaraa/sql-data-analytics-project)

Make sure both the required software is installed and configured properly before proceeding.
